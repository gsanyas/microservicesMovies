# User component

Composant qui implémente les fonctionnalités CRUD des users

## Install

`npm install`

## Define environment variables

Replace the values by your configuration
```[Text]
ORIGIN=http://localhost:8000
EXPRESS_PORT=8002
```

In a local installation, write it in a .env file.

## Run

`npm run start`

## Test using jest

`npm run test`
